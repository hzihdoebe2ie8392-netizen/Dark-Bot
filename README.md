# Dark Bot 🛡️

بوت إدارة وحماية مجموعات واتساب احترافي مبني على Baileys.

---

## المتطلبات

- Node.js 20+
- npm
- PM2 (للتشغيل المستمر)
- Ubuntu / Oracle Cloud

---

## التثبيت

```bash
# 1. فك الضغط
unzip Dark-Bot.zip
cd Dark-Bot

# 2. تثبيت الحزم
npm install

# 3. إعداد ملف البيئة
cp .env.example .env
nano .env

# 4. تشغيل البوت لأول مرة لمسح الـ QR
node src/index.js

# 5. بعد مسح الـ QR والاتصال، أوقف البوت (Ctrl+C)

# 6. تشغيل مع PM2
npm install -g pm2
pm2 start ecosystem.config.js
pm2 save
pm2 startup
```

---

## إعداد ملف .env

```env
OWNER_NUMBER=201107897471       # رقم المالك بدون + أو 00
BOT_NAME=Dark Bot
SESSION_NAME=dark-bot-session
DB_PATH=./data/dark_bot.db
LOG_LEVEL=info
SPAM_LIMIT=3                    # عدد الرسائل المكررة قبل اعتبارها سبام
SPAM_MUTE_DURATION=3600         # مدة الكتم بالثواني (3600 = ساعة)
WARN_LIMIT=3                    # عدد التحذيرات قبل الكتم التلقائي
```

## أوامر الإدارة

| الأمر | الوصف |
|-------|-------|
| طرد | طرد عضو |
| كتم | كتم عضو |
| فك كتم | رفع الكتم |
| رفع | ترقية لمشرف |
| تنزيل | تنزيل من مشرف |
| تحذير | إعطاء تحذير |
| عرض التحذيرات | عرض تحذيرات عضو |
| تصفير التحذيرات | مسح تحذيرات عضو |
| قفل الشات | قفل المجموعة للمشرفين فقط |
| فتح الشات | فتح المجموعة للجميع |
| تثبيت | تثبيت رسالة |
| فك تثبيت | إلغاء تثبيت رسالة |

## أوامر الحماية

| الأمر | الوصف |
|-------|-------|
| قفل الروابط / فتح الروابط | منع/السماح بالروابط |
| قفل الصور / فتح الصور | منع/السماح بالصور |
| قفل الفيديو / فتح الفيديو | منع/السماح بالفيديو |
| قفل الملفات / فتح الملفات | منع/السماح بالملفات |
| قفل الملصقات / فتح الملصقات | منع/السماح بالملصقات |
| قفل الصوتيات / فتح الصوتيات | منع/السماح بالصوتيات |
| قفل الجيفات / فتح الجيفات | منع/السماح بالجيفات |
| قفل التوجيه / فتح التوجيه | منع/السماح بالرسائل المعادة |

## أوامر المالك (في الخاص فقط)

| الأمر | الوصف |
|-------|-------|
| المالك | لوحة التحكم |
| إذاعة | إرسال رسالة لكل الجروبات |
| الجروبات | قائمة المجموعات |
| صيانة | تفعيل وضع الصيانة |
| إيقاف الصيانة | تعطيل الصيانة |
| نسخة احتياطية | إنشاء backup |
| إعادة تشغيل | إعادة تشغيل البوت |
| حظر [رقم] | حظر عالمي |
| فك حظر [رقم] | إلغاء الحظر |

## نصائح للأداء على Oracle Cloud

```bash
# زيادة file descriptors
ulimit -n 65536

# تشغيل مع PM2 (موصى به)
pm2 start ecosystem.config.js
pm2 startup systemd
pm2 save
```

## السجلات

السجلات محفوظة في مجلد `logs/` بتنسيق يومي.

```bash
# عرض سجلات PM2
pm2 logs dark-bot

# عرض سجلات اليوم
cat logs/$(date +%Y-%m-%d).log
```
